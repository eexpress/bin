grep写的复杂，只是为了结果好看些。
```
▶  sudo dmidecode --type memory | grep -e 'Maximum Total Memory Size' -e Slots -e 'Data Width' -e '^\s*Size' -e DDR -e '^\s*Speed' -e Manufacturer | grep "[0-9]*"
        Maximum Total Memory Size: 8192 MB
        Associated Memory Slots: 1
        Data Width: 64 bits
        Size: 4096 MB
        Type: DDR3
        Speed: 1600 MHz
        Manufacturer: Hynix
```
